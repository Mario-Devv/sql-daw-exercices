Ejercicios para prácticar lo visto en clase, empezamos por enunciados básicos
Un videoclub quiere gestionar el alquiler de sus películas:
- El videoclub tiene películas disponibles. De cada película se guarda: código de película, título, director, año de estreno y género.
- El videoclub tiene socios registrados. De cada socio se registra: número de socio, nombre, apellidos y fecha de alta.
- Un socio puede alquilar varias películas a lo largo del tiempo.
- Una película puede ser alquilada por diferentes socios en distintos momentos.
- De cada alquiler se necesita registrar: fecha de alquiler y fecha de devolución.
